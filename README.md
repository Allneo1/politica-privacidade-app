Política de Privacidade - All List

Este aplicativo respeita sua privacidade e está empenhado em proteger seus dados.

Coleta e armazenamento de informações:
Dados inseridos pelo usuário: O aplicativo armazena localmente em seu dispositivo os títulos e horários dos lembretes e itens da lista que você cria. Esses dados não são enviados para nossos servidores nem compartilhados com terceiros.

Dados Não Pessoais: Podemos coletar informações não identificáveis, como modelo do dispositivo, versão do sistema operacional e identificadores de publicidade para fins de análise e funcionamento técnico.

Não lunático e alarmes:
Este aplicativo solicita permissões para enviar notificações e agendar alarmes exatos. As permissões são permitidas apenas para garantir que os lembretes configurados por você sejam disparados no horário correto. O conteúdo das notificações é processado dentro do seu dispositivo.

Anúncios:
Este aplicativo utiliza o Google AdMob para exibição de anúncios. O Google pode utilizar cookies ou identificadores de publicidade anônimos para personalizar os anúncios exibidos.

Serviços de Terceiros: O aplicativo utiliza serviços que possuem políticas de privacidade próprias:
Serviços do Google Play

Google AdMob

Dados de Crianças: Não coletamos intencionalmente dados pessoais de crianças. Como o armazenamento é local, os pais podem gerenciar ou usar excluindo os dados do aplicativo.

Alterações e Contato: Esta política pode ser atualizada periodicamente. Para dúvidas, entre em contato via: Email: balttechnologia@gmail.com
